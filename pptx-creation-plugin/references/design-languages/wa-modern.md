# Design language: wa-modern（和モダン）

Refined Japanese-modern. An 藍 (indigo) and 墨 (sumi) base on 和紙-warm paper, a
朱 (vermilion) accent, and a **Mincho** heading — quiet, grounded, unmistakably 和.

- **Tokens** (`themes/wa-modern/theme.json`): warm off-white `F7F4EE`, indigo-ink
  `1F2A38`, indigo dark `1B2A3A`; vermilion accent `A6432E`; airy leading. Font:
  **heading = Yu Mincho (serif)**, body = Yu Gothic Medium.
- **Feels:** composed, premium, distinctly Japanese without being retro.
- **Use for:** Japanese corporate/brand decks, hospitality, craft/heritage
  businesses, mission/vision openers with a 和 sensibility.
- **Avoid for:** hard data readouts (use `data-driven`), Western startup pitches
  (use `swiss`/`hybrid-editorial`).

The vermilion is an accent (dot, kicker, one card), never a fill band. Requires
Yu Mincho (setup.sh registers it). See [`../../themes/wa-modern/theme.json`].
