# Design language: neutral-business

The quiet floor and the default. A calm dark-navy + single teal accent, cards and
type doing the work, no imagery. Everything else on the shelf is measured against
this — it is also the reference render (`examples/seminar-kanrikaikei/`).

- **Tokens** (`themes/_default-neutral/theme.json`): white body / dark-navy
  sandwich, one teal accent `0E7C86`; Yu Gothic weight-led hierarchy; modular
  ~1.26 scale; margin `0.75`. Image slots left empty (image-0).
- **Feels:** trustworthy, understated, decision-first.
- **Use for:** board packs, lenders/investors, financial and internal reviews —
  anywhere the numbers matter more than the mood, and images would distract.
- **Avoid for:** marketing/brand openers that need presence (use `hybrid-editorial`
  / `editorial`) — neutral can read as plain when warmth is the goal.

This is the language the whole floor was tuned on; when unsure, start here.
See [`../../themes/_default-neutral/theme.json`].
