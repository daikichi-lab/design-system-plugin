# Design language: swiss

International Typographic Style. Objective, gridded, generous whitespace, a
strong flush-left hierarchy, and **one loud accent** (Swiss red). Nothing
decorative — the grid and the type do the work.

- **Tokens** (`themes/swiss/theme.json`): near-black ink on white; a single red
  accent `D6001C`; large display type (cover 48 / title 32); wide margin `0.9`;
  tight heading leading. Font: **BIZ UDGothic** (clean universal-design gothic).
- **Feels:** precise, confident, timeless. The red is used sparingly — a dot, a
  kicker, one emphasized card — never as fill everywhere.
- **Use for:** consultancies, product/strategy decks, anything that should read
  as rigorous and modern.
- **Avoid for:** warm/soft brand stories (use `editorial`/`wa-modern`), or
  data-dense readouts where the whitespace wastes room (use `data-driven`).

Governed by the whole floor (Phase A type, Phase B kinsoku, the AI-tell blocklist
— the red is an accent, never a stripe). See [`../../themes/swiss/theme.json`].
